# AGGREGATION-TASK
Property24 Data Analysis (SQL Project)

Overview

This project demonstrates the use of SQL aggregation functions to analyze a simulated Property24 real estate dataset. The analysis was performed using SQL Server Management Studio (SSMS).

The main objective was to explore property data and extract insights such as total listings, average prices, and overall property values.

---

 Tools & Technologies

- SQL
- SQL Server Management Studio (SSMS)

---

 Key Features

- Data analysis using SQL queries
- Use of aggregation functions:
  - "COUNT()"
  - "SUM()"
  - "AVG()"
- Data grouping using "GROUP BY"
- Simple and clear query structure for analysis



🧠 What I Learned

- How to analyze structured datasets using SQL
- How to summarize data using aggregation functions
- How to group data to uncover trends and patterns
- Writing clean and readable SQL queries
